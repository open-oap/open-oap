OAP Backup Tool

Create Complete Backup

With this tool you can create a complete backup of your TYPO3 installation. The backup includes:

  • All files of the TYPO3 installation
  • The complete database
Note: Creating the backup may take some time depending on the size of your installation. The process runs in the background and you will be notified by email once the backup is completed.
